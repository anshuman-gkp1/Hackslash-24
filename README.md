# 🏆 Hackslash-24 — "Let's Collaborate" Website Clone

A **hackathon project** from Hackslash 2024, where the task was to clone the UI of the [Let's Collaborate](https://lets-collab-rust.vercel.app/) business website. Built using pure **HTML and CSS** with a dark/night mode toggle feature.

## 🌟 Features

- 🎨 **Pixel-perfect UI clone** of a professional services website
- 🌙 **Night Mode Toggle** — Switch between light and dark themes with a button click
- 📐 **Responsive Layout** — Sections built using CSS Flexbox
- 🖼️ **Dynamic Image Sections** — Four service panels: AI + RPA, App Development, DevOps, Cloud
- 🔤 **Google Fonts** — Uses Roboto Slab for clean typography

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure and content |
| CSS3 | Layout, styling, and dark mode |
| Vanilla JavaScript | Night mode toggle logic |
| Google Fonts | Typography (Roboto Slab) |

## 🚀 How to Run

No installation needed — it's a static HTML page!

1. Clone the repository:
   ```bash
   git clone https://github.com/anshuman-gkp1/Hackslash-24.git
   ```
2. Open `hackslash/website/clone.html` in any browser
3. Click **"Toggle Night Mode"** to switch themes

Hackslash is a web development hackathon that challenges participants to clone or recreate real-world web interfaces under time constraints, testing HTML/CSS proficiency and attention to design detail.
